# Önkoşullar

## Gerekli

- **Node.js 18 veya üzeri** - Shevky, `@shevky/core` motor gereksinimlerinde belirtildiği gibi Node.js 18+ sürümünü gerektirir.
- **npm** - Node.js ile birlikte gelir. Paketleri yüklemek ve komut dosyalarını çalıştırmak için kullanılır.

## Ortamınızı Doğrulayın

```bash
node --version   # must be v18.x or later
npm --version    # any recent version
```

## İsteğe bağlı

- **Git** - sürüm kontrolü istiyorsanız veya `--init` bir başlangıç ​​şablonunu kopyalıyorsa gereklidir.
- **Bir kod düzenleyici** - VS Code veya Markdown ve JSON desteğine sahip herhangi bir düzenleyici.
