# Eklenti Kataloğu

Bu bölümde her bir first-party eklenti için amaç, hook'lar, yapılandırma, bağımlılıklar ve bilinen riskler belgelenir.

## Eklentiler

1. [Robots.txt](./plugin-robots-txt/) - `robots.txt` üretir
2. [Tailwind CSS](./plugin-tailwindcss/) - Tailwind CSS çıktısını derler
3. [esbuild](./plugin-esbuild/) - JavaScript bundle üretir
4. [RSS](./plugin-rss/) - dil bazlı RSS beslemeleri üretir
5. [Sitemap](./plugin-sitemap/) - `sitemap.xml` üretir
6. [Content Bridge](./plugin-content-bridge/) - harici API içeriğini içeriğe dönüştürür
7. [Open Graph](./plugin-open-graph/) - Open Graph, Twitter Card ve JSON-LD meta verilerini üretir
