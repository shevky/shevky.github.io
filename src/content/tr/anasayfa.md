# S.H.E.V.K.Y.

**S.H.E.V.K.Y.** (Static HTML Engine via Keyed YAML) bir statik site üreticisi. Markdown dosyaları ve static dosyaları alır ve hepsini bir statik web sayfası olarak size teslim eder. Sitenin görünümüz ve Java Script ile yapılacak etkileşimleri size kalmış.

![S.H.E.V.K.Y. Process Flow](/assets/images/shevky-process-flow.webp)

Düşük boyutlu ve işlevsel statik web sayfaları oluşturmak için biçilmiş kaftandır.

## Önkoşullar

**S.H.E.V.K.Y.** NodeJS ile geliştirilmiş bir CLI aracıdır. Makinenizde NodeJS v20 veya üstü bir sürüm olması yeterlidir.
