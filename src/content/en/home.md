# S.H.E.V.K.Y.

**S.H.E.V.K.Y.** (Static HTML Engine via Keyed YAML) is a static site generator. It takes Markdown files and static assets and delivers them as a complete static website. The visual design and any JavaScript interactions are up to you.

![S.H.E.V.K.Y. Process Flow](/assets/images/shevky-process-flow.webp)

It is built for lightweight, functional static websites.

## Prerequisites

**S.H.E.V.K.Y.** is a CLI tool built with NodeJS. You only need NodeJS v20 or later on your machine.
